# Movie Rating App

This project is separated into two parts:

- `client`: The frontend React application.
- `server`: The backend Node.js/Express application.

## How to run

### Client
1. `cd client`
2. `npm install` (if not already installed)
3. `npm run dev`

### Server
1. `cd server`
2. `npm install` (if not already installed)
3. `npm start`
